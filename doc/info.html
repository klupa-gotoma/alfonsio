Alfons.io — The Things Stack Integration

Connect your LoRaWAN devices from The Things Stack directly to Alfons.io.

Webhook Setup

In The Things Stack Console go to Integrations → Webhooks → Add webhook → Choose “Alfons.io”, then enter your API Token.

Webhook endpoint:

https://portal.alfons.io/v3/capture?secret=YOUR_API_TOKEN

Example Test

Send a test uplink to verify connection:

curl -X POST "https://portal.alfons.io/v3/capture?secret=abcd1234" \
  -H "Content-Type: application/json" \
  -d '{"dev_id":"device123","payload":"AQID","metadata":{"time":"2025-10-03T08:00:00Z"}}'

Example Payload

{
  "end_device_ids": {
    "device_id": "device123"
  },
  "uplink_message": {
    "f_port": 1,
    "frm_payload": "AQID",
    "decoded_payload": { "temperature": 23.4 }
  },
  "received_at": "2025-10-03T08:00:00Z"
}
